# Risk Register

## Data Risks

### Risk: Incorrect Data Entry
Impact:
- Wrong risk calculations
- Incorrect interventions

Mitigation:
- Form validation
- Required fields
- Input constraints

---

### Risk: Missing Data
Impact:
- Incomplete assessment

Mitigation:
- Validation checks
- Missing-value warnings

---

## Security Risks

### Risk: Unauthorized Access
Impact:
- Exposure of sensitive information

Mitigation:
- Role-based permissions
- Authentication
- Audit logging

---

## Operational Risks

### Risk: System Failure
Impact:
- Users cannot access the platform

Mitigation:
- Error handling
- Logging
- Backup procedures

---

### Risk: Silent Workflow Failure
Impact:
- Cases not processed correctly

Mitigation:
- Alerts
- Dashboard monitoring
- Audit trail

---

## Decision Risks

### Risk: False Positive Alert
Impact:
- Unnecessary intervention

Mitigation:
- Human review before action

---

### Risk: False Negative Alert
Impact:
- At-risk student not detected

Mitigation:
- Periodic reassessment
- Multiple indicators
