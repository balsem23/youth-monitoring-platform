# Roles Matrix

| Action | Admin | Teacher | Counselor |
|---|---|---|---|
| View dashboard | Yes | Yes | Yes |
| Create student case | Yes | Yes | No |
| Update education data | Yes | Yes | No |
| View risk alerts | Yes | Yes | Yes |
| Create intervention plan | Yes | No | Yes |

| Manage users/settings | Yes | No | No |

## Access Control Rule

Each role must only access actions related to its responsibility. Unauthorized actions must be blocked and logged.
